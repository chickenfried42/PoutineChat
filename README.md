# POUTINECHAT!!
PoutineChat is an arbitrarily named chat website with customizability. Discuss food or whatever you want with anyone you find!
Will mostly get regular updates every few days.

https://poutinechat.web.app/

# Features
Message People - Talk to whoever's online in your current chat room about anything, like food.<br>
Hop to different rooms - Tired of chatting about a certain topic (like food)?, simply switch to a different chatroom using /room<br>
Helpful Manual - A compilation of all of PoutineChat's commands<br>
Cool themes - Features four different color schemes with more being added<br>
Customization commands - Adjust the size of user icons or the space each message takes up to your liking<br>
ZERO AGE VERIFICATION!!!!!

# Tech Stack
HTML, JS (vanilla), CSS<br>
Database, Authentication: Firebase
Deployment: Firebase
